# Trợ lý Quản lý Lịch trình Cá nhân (Personal Schedule Assistant)

## 1. Giới thiệu
Ứng dụng quản lý lịch trình tích hợp xử lý ngôn ngữ tự nhiên tiếng Việt, giúp người dùng thêm sự kiện nhanh chóng bằng văn bản tự do.

## 2. Cài đặt
1. Yêu cầu: Python 3.8+
2. Cài đặt thư viện:
   `pip install -r requirements.txt`
3. Chạy ứng dụng:
   `python src/main.py`

## 3. Tính năng chính
- Nhập liệu bằng câu tiếng Việt tự nhiên.
- Tự động trích xuất: Tên sự kiện, Thời gian, Địa điểm.
- Nhắc nhở qua Pop-up hệ thống.
- Lưu trữ cục bộ (SQLite).
